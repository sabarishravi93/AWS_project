# AWS Project - KodeKloud Engineer Challenges

This repository contains hands-on AWS lab solutions completed as part of the KodeKloud Engineer learning path. Each entry is a daily task that focuses on key AWS services and best practices for infrastructure, security, automation, and operations.

## Summary
- 50 days of AWS exercises
- Covers EC2, VPC, IAM, S3, RDS, Lambda, ECS, EKS, CloudFormation, monitoring, networking, and security
- Objective: build repeatable, secure, and scalable AWS workflows

## How to use
- For each day, open the corresponding `100_Days_of_Cloud-AWS/DayXX.md` file
- Follow steps in sequential order for learning progression
- Reuse scripts and CloudFormation templates where possible

## Notes
- All AWS resources should be cleaned up after completion
- Apply least privilege security for IAM policies and roles
- Validate with AWS CLI and CloudWatch as required
